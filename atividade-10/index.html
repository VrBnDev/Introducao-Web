Propriedade display

A propriedade display diz respeito a como determinado elemento vai ser renderizado. Essa propriedade pode assumir os valores:

  • block
  • inline
  • none
  • inline-block
  • list-item
  • run-in
  • table

display:block

Renderiza o elemento como bloco.

Primeiro {display:block} Segundo {display:block} Terceiro {display:block}

display:inline

Renderiza o elemento em linha.

display:block
display:inline
display:block

display:none

O valor none da propriedade display torna o elemento oculto. Abaixo existem três elementos, porém só dois são visiveis.

display:block
display:inline
display:none

display:inline-block

Com o inline-block é possível definir altura e largura ao elemento, e adicionar quebra de linha no meio de um paragráfo.
